ВНИМАНИЕ! 
Прошивка созданна в среде Visual Studio Code + PlatformIO
Для обновления прошивки под свою существующую плату требуется только скопировать папку \src
 
Для работы и компиляции в среде Arduino IDE:
- требуется воспользоваться файлами находящимися в папке \src

ВАЖНО! 
На данный момент это предрелизная версия прошивки, лишенная некоторых возможностей.
Изначально концепция была направлена на переосмысливание некоторых составных блоков, и пока они не опробованы, они отключены. 
В дальнейшем планируется полное восстановление функций исходной прошивки Gunner47 при значительном увеличении качества кода и потребительских свойств Лампы. 
Пока еще не рабоатет (в отличие от оригинальной прошивки):
- бегущая строка для вывода времени и событий
- фаворитные списки
- будильник Рассвет
- таймер отключения
- ...

Файл конфигурации позволяет с гибкостью настроивать Лампу под свои нужды. Но, практика не может обеспечить совместимость листов конфигураций авторского и пользовательского проекта.
Так как, приходится активно пополнять и видоизменять данный файл (constants.h), то для обеспечения уже с существующими проектами описания переменных (настроек) разбито по разделам с датой ревизии (изменения).
Так легче отследить факт изменений в уже созданном пользовательском файле конфигурации.

ОЧЕНЬ ВАЖНО!
При первой конфигурации обратите внимание на следующие моменты:
// ----- MATRIX / МАТРИЦА ----- 
#define PREINDICATORS (0U)      - должно быть 0! если вы не используете светодиоды ленты как индикаторы
#define POSTINDICATORS (0U)     - должно быть 0! если вы не используете светодиоды ленты как индикаторы

- Развертывание менеджера подключения к сети происходит только тогда, когда ESP_MODE=1 и НАЖАТА КНОПКА ПРИ ВКЛЮЧЕНИИ ЛАМПЫ!
- Прошивка работает на всех последних версиях используемых библиотек! (возможен конфликт со старыми версиями прошивок)
